# TireHub - Tire Company Website

A modern, responsive single-page website for a tire company built with React and Tailwind CSS.

## Features

- **Responsive Header Navigation** - Clean navigation bar that works on all devices
- **Hero Section** - Eye-catching hero with call-to-action button
- **Tire Search Filters** - Search tires by width, aspect ratio, and diameter
- **Product Showcase** - Display tire products in an elegant card layout

## Technologies Used

- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)

## Installation

1. Clone the repository:

```bash
   git clone https://github.com/YOUR-USERNAME/tire-company-website.git
```

2. Navigate to the project directory:

```bash
   cd tire-company-website
```

3. Install dependencies:

```bash
   npm install
```

4. Run the development server:

```bash
   npm run dev
```

5. Open your browser and visit `http://localhost:5173`
